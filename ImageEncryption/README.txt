Click on ImageEncryption.jar to run the application.
Current application has a GUI that loads images when a file is chosen. 
Once cover and original images are loaded, encryption can be done using 'Encrypt Image' button. 
Intermediate steps are shown as images, along with the final encrypted image. 
Once encryption is complete, 'Save File' dialog box pops up to allow user to save encrypted image.
